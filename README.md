# 🕺 Real-Time Body Pose Detection Game Controller

Control games using your body movements with real-time pose detection powered by Machine Learning.

---

## 🚀 Project Overview

This project uses **computer vision and machine learning** to detect human body movements through a webcam and convert them into game controls.

You can:

* Start the game by **joining both hands**
* Control gameplay using **body gestures**
* Play browser-based games like running games using only your body

---

## 🎯 Features

* 🧠 Real-time pose detection using ML
* 🎮 Control games without keyboard
* ✋ Gesture-based interaction
* ⚡ Lightweight and fast processing
* 📷 Works with standard webcam

## 📦 Installation

### 1. Clone the repository

```bash
git clone - https://github.com/ashrithavarmaa661/Real_time_body_pose_detection.git
cd Real_time_body_pose_detection
```

### 2. Create virtual environment

```bash
python -m venv .venv
```

### 3. Activate environment

**Windows:**

```bash
.venv\Scripts\activate
```

### 4. Install dependencies

```bash
python -m pip install mediapipe==0.10.9 opencv-python==4.8.1.78 numpy==1.24.4 pyautogui
```

---

## ▶️ How to Run

1. Open any browser game (like running games)
2. Click on the game window
3. Run the Python script:

```bash
python RTBP.py
```

---

## 🎮 Controls

| Body Movement   | Action     |
| --------------- | ---------- |
| Join both hands | Start game |
| Both hands up   | Jump       |
| Left hand up    | Move left  |
| Right hand up   | Move right |
| Bend down       | Slide      |

* Requires clear full-body visibility
* Browser games must support keyboard input
